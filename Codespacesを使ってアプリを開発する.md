# 前提
- main branchを直接編集することがないように注意してください。（branch はCodespaes を開いて左下に表示があります。）
- Codespaces は30分間未操作が続くと停止されます。挙動が不安定なときはCodespaces を開いているタブをリロード（更新）、または閉じてから再度Codespaces を開いてください。

# Codespaces を使って作ったアプリケーションをブラウザで確認する方法
1. 拡張機能「liveserver」をインストールする
2. 右下にある「Go Live」をクリックして、ポートを開放すればブラウザでアプリケーションが立ち上がる

- ワークショップを進める上で、安定した動作確認とスピード感を保つために上記構成と、HTML, CSS, JavaScriptの構成を推奨します。
  - 別言語、フレームワークの利用は問題ありませんが、チーム全員が共通して動作・開発ができることが求められます。トラブルシューティングも基本的にはチーム内ですべて行います。
